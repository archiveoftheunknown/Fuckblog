TO SET UP IN NEW REPOSITORY:
-----------------------------

1. Extract the archive:
   tar -xzf complete-political-insights-project.tar.gz

2. Install dependencies:
   npm install

3. Run development server:
   npm run dev
   (This starts both frontend and backend)

4. Build for production:
   npm run build

5. The dist/public folder contains the built files for deployment

FOR GITHUB PAGES DEPLOYMENT:
-----------------------------
After building, use the files in dist/public/:
- Upload all files from dist/public to your GitHub repo
- Enable GitHub Pages from Settings > Pages
- Select main branch and root folder
- Your site will be live at https://[username].github.io/[repo-name]/
